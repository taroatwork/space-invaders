# Space Invaders

クラシックな1978年のアーケードゲーム「スペースインベーダー」をWebで再現したゲームです。

## 遊び方

### PC版
- **左右移動**: ← → キー または A D キー
- **発射**: スペースキー または ↑ キー（押し続けで連射）
- **ポーズ**: P キー
- **リスタート**: ゲームオーバー時にスペースキー

### モバイル版
- **左右移動**: 画面下部の ◀ ▶ ボタン
- **発射**: FIRE ボタン（押し続けで連射）
- **ポーズ**: ゲーム画面をタップ
- **リスタート**: ゲームオーバー時に FIRE ボタン

### ゲームルール
- インベーダーを全滅させるとステージクリア
- インベーダーが地上に到達するとゲームオーバー
- ライフは3つ、インベーダーのミサイルに当たると1つ減少
- 防御壁（シールド）は自機とインベーダーの両方の攻撃で破壊される
- UFO（赤い飛行物体）を撃墜すると高得点

### 得点
- 小型インベーダー（最上段）: 30点
- 中型インベーダー（2-3段目）: 20点
- 大型インベーダー（4-5段目）: 10点
- UFO: 50〜300点（ランダム）

## 技術スタック

- **言語**: TypeScript
- **ビルドツール**: Vite
- **レンダリング**: HTML5 Canvas（5倍解像度）
- **オーディオ**: Web Audio API
- **PWA**: vite-plugin-pwa + Workbox
- **デプロイ**: Vercel

## プロジェクト構成

```
src/
├── main.ts              # エントリーポイント
├── types/               # 型定義
├── game/
│   ├── Game.ts          # メインゲームクラス
│   ├── entities/        # ゲームエンティティ
│   │   ├── Player.ts    # 自機
│   │   ├── Invader.ts   # インベーダー
│   │   ├── Bullet.ts    # 自機の弾
│   │   ├── Missile.ts   # インベーダーの弾
│   │   ├── Shield.ts    # 防御壁
│   │   └── UFO.ts       # UFO
│   └── utils/
│       └── constants.ts # ゲーム定数
├── rendering/
│   └── Renderer.ts      # Canvas描画
└── audio/
    └── AudioManager.ts  # サウンド管理
```

## 開発

### セットアップ
```bash
npm install
```

### 開発サーバー起動
```bash
npm run dev
```

### ビルド
```bash
npm run build
```

### モバイル実機テスト
```bash
npm run dev -- --host
```

## 機能

- オリジナルに忠実なドット絵スプライト
- 4音ベースラインによるインベーダー移動音
- インベーダー残数に応じた加速（オリジナルの「バグ」を再現）
- ピクセル単位のシールド破壊
- 3種類のミサイルパターン（直進/高速/蛇行）
- PWA対応（オフライン動作、ホーム画面に追加可能）
- iOS Safari/Chrome対応のオーディオ
